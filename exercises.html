Naive BayesUAS 2023/24 · Soal 1

1. Naive Bayes Assumption

Which of the following statements correctly describes the Naive Bayes assumption for features $x_1, x_2, x_3$ and class label $y$? (Select TRUE statements.)

  • A. $x_1, x_2, x_3$ are unconditionally independent of one another.
  • B. If $y$ is known, $x_1, x_2, x_3$ are independent of each other.
  • C. $P(x_1, x_2, x_3 \mid y) = P(x_1 \mid y) P(x_2 \mid y) P(x_3 \mid y)$.
  • D. $P(y \mid \mathbf{x}) = P(y)P(\mathbf{x} \mid y) / P(\mathbf{x})$ (this is the NB assumption).
Solution

Correct: B and C.

The NB assumption is conditional independence given the class — not unconditional independence. So A is too strong (false). D is just Bayes' theorem, which holds always — it isn't the NB assumption itself.

The factorization $P(x_1, x_2, x_3 \mid y) = \prod_j P(x_j \mid y)$ in option C is the mathematical expression of conditional independence, which is exactly the NB assumption.

Encoding + ModelsUAS 2023/24 · Soal 2

2. One-hot encoding and model choice

Features: weight (1 numeric), height (1 numeric), gender (categorical, 2 values), age (1 numeric), catheter type (categorical, 3 values), duration (1 numeric). What is the total length of $\mathbf{x}$ after one-hot encoding?

For three target variables:

  • $y_1$ = infected (yes/no)
  • $y_2$ = severity (low/medium/high)
  • $y_3$ = number of recovery days

What model and parameter shapes are appropriate?

Solution

Length of $\mathbf{x}$: $1 + 1 + 2 + 1 + 3 + 1 = 9$. Contains binary elements from the one-hot encoding.

Model choices:

  • $y_1$ (binary): logistic regression, $y_1 = \sigma(\mathbf{w}^\top \mathbf{x} + b)$, $\mathbf{w}$ length 9, $b$ scalar.
  • $y_2$ (3 classes): softmax regression, $\mathbf{W} \in \mathbb{R}^{3 \times 9}$, $\mathbf{b} \in \mathbb{R}^3$.
  • $y_3$ (continuous): linear regression, $y_3 = \mathbf{w}^\top \mathbf{x} + b$, $\mathbf{w}$ length 9, $b$ scalar.
Gradient DescentUAS 2023/24 · Soal 3

3. Gradient Descent — correct statements

For the update $\mathbf{w}_b = \mathbf{w}_a - \alpha \nabla E(\mathbf{w}_a)$, which statements are TRUE?

  • A. $(\mathbf{w}_b - \mathbf{w}_a)$ points in the opposite direction of $\partial E/\partial \mathbf{w}_a$.
  • B. If $\|\mathbf{w}_b - \mathbf{w}_a\| \to 0$, we are approaching a local minimum.

MLEUAS 2023/24 · Soal 4

4. Bernoulli likelihood

Data $\mathcal{D} = (1, 1, 1, 0, 0)$ — 3 successes, 2 failures. Which statements are TRUE?

  • A. $L(p) = p^3(1-p)^2$.
  • B. $L$ achieves its maximum when $p = 3/5 = 0.6$.
  • C. We estimate $p$ by finding the minimum of $L$.
Solution

Correct: A and B. Statement C is false — MLE seeks the maximum, not the minimum, of the likelihood.

Verification: $LL(p) = 3\log p + 2\log(1-p)$. Set $\frac{dLL}{dp} = \frac{3}{p} - \frac{2}{1-p} = 0 \Rightarrow 3(1-p) = 2p \Rightarrow p = 3/5 = 0.6$. ✓

Select the TRUE statements about L1 and L2 regularization:

  • A. L2 regularization adds $\lambda \mathbf{w}^\top \mathbf{w}$ with $\lambda$ a hyperparameter.
  • B. If $\lambda$ on L2 is very large, all parameters tend toward 0.

MLP ActivationUAS 2023/24 · Soal 6

6. MLP activation functions

Which statements are TRUE?

  • A. If all layers use linear activation, MLP is equivalent to linear regression.
  • B. Hidden linear + 1 output sigmoid is equivalent to logistic regression.
  • C. Softmax outputs at the final layer are independent of each other.
  • D. Output activation is chosen without considering the task.
Solution

Correct: A and B only. Composing linear maps gives a linear map; linear hidden + sigmoid output = logistic regression.

C is false: softmax outputs are not independent — the denominator normalizes across ALL units. D is also false: output activation is task-dependent (sigmoid for binary, softmax for multi-class, identity for regression).

Cross-entropyUAS 2023/24 · Soal 7

7. Softmax cross-entropy

Softmax output $\mathbf{p} = [0.2, 0.1, 0.5, 0.2]$, target one-hot $\mathbf{t} = [1, 0, 0, 0]$ (class A). What is the cross-entropy loss?

  • A. $-\log(0.5) \approx 0.693$
  • B. $-\log(0.1) \approx 2.303$
  • C. $-\log(0.2) \approx 1.609$
  • D. $-(0.2 + 0.1 + 0.5 + 0.2) = -1$
Solution

$L = -\sum_k t_k \log p_k$. Since only $t_1 = 1$:

$$L = -(1 \cdot \log 0.2 + 0 + 0 + 0) = -\log(0.2) \approx 1.609.$$

Cost-sensitiveUAS 2023/24 · Soal 9

9. Balanced class weighting

Confusion matrix (rows = actual, cols = predicted): Low → [102, 898], High → [48, 2]. Total $N = 1050$. The "High" class has $48 + 2 = 50$ samples; "Low" has 1000. What is the balanced weight for the "High risk" class?

  • A. 0.525
  • B. 21
  • C. 10.5
  • D. 50
Solution

Balanced weight formula: $w_k = \dfrac{N}{K \cdot n_k}$.

$$w_{\text{High}} = \frac{1050}{2 \cdot 50} = \frac{1050}{100} = 10.5.$$

Sanity-checking other options: accuracy is NOT a good metric on imbalanced data; total misclassification cost = $48 \cdot 25 + 898 = 2098$ (not 1200); F1 here is tiny (≈ 0.004), not 0.57.

K-MeansUAS 2023/24 · Soal 10

10. K-Means with Manhattan distance

Points: A1(2,10), A2(8,4), A3(1,2), A4(6,4), A5(4,9). Initial centroids: $C_1 = (2,10)$, $C_2 = (1,2)$. Use Manhattan distance. After the FIRST iteration, $C_1$ consists of which points?

  • A. {A1, A2}
  • B. {A1, A5}
  • C. {A1, A4, A5}
  • D. {A1 only}
Solution

Manhattan distances to $(C_1, C_2)$:

  • A1(2,10): (0, 9) → $C_1$ closer
  • A2(8,4): (12, 9) → $C_2$ closer
  • A3(1,2): (9, 0) → $C_2$ closer
  • A4(6,4): (10, 7) → $C_2$ closer
  • A5(4,9): (3, 10) → $C_1$ closer

So in the first iteration: $C_1 = \{A1, A5\}$, $C_2 = \{A2, A3, A4\}$.

Naive Bayes (mixed)UAS 2023/24 · Soal 11

11. Naive Bayes with categorical + Gaussian features

Dataset of 16 rows: 7 cotton, 9 soybean. Predict crop for query R=high rainfall, H=low humidity, T=24°C. Conditional probabilities given:

  • $P(R=\text{high} \mid \text{cotton}) = 2/7 = 0.2857$; $P(R=\text{high} \mid \text{soy}) = 5/9 = 0.5556$.
  • $P(H=\text{low} \mid \text{cotton}) = 0.1000$ (after Laplace); $P(H=\text{low} \mid \text{soy}) = 0.2500$.
  • Gaussian on $T = 24$: $f_{\text{cotton}}(24) = 0.0676$; $f_{\text{soy}}(24) = 0.0316$.

Which crop is more likely?

  • A. Cotton
  • B. Soybean
Solution

Class priors: $P(\text{cotton}) = 7/16 = 0.4375$; $P(\text{soy}) = 9/16 = 0.5625$.

Unnormalized posteriors:

$$\text{cotton}: 0.4375 \times 0.2857 \times 0.1000 \times 0.0676 \approx 0.0008,$$ $$\text{soy}: 0.5625 \times 0.5556 \times 0.2500 \times 0.0316 \approx 0.0025.$$

Since $0.0025 > 0.0008$, soybean is the more likely classification.

MLP forward+backUAS 2023/24 · Soal 12

12. MLP forward & backward pass

Architecture: $\mathbf{W}^{(1)}$ is $5 \times 4$, $\mathbf{W}^{(2)}$ is $3 \times 5$. From this:

  • Input features = ? Hidden units = ? Output units = ?
  • Task type?
  • If forward pass gives logits $\mathbf{z}^{(3)} = [1.306, 2.33, -0.68]$, what is $\mathbf{a}^{(3)} = \text{softmax}(\mathbf{z}^{(3)})$?
Solution

Architecture: 4 inputs (cols of $\mathbf{W}^{(1)}$), 5 hidden units (rows of $\mathbf{W}^{(1)}$, cols of $\mathbf{W}^{(2)}$), 3 output units (rows of $\mathbf{W}^{(2)}$). Task: 3-class classification.

Softmax:

  • $e^{1.306} \approx 3.691$, $e^{2.33} \approx 10.278$, $e^{-0.68} \approx 0.507$. Sum $\approx 14.476$.
  • $a_1^{(3)} \approx 3.691/14.476 = 0.255$
  • $a_2^{(3)} \approx 10.278/14.476 = 0.710$
  • $a_3^{(3)} \approx 0.507/14.476 = 0.035$

Sums to 1.0 ✓. The most likely class is class 2.

Backprop highlights (assuming softmax + CCE so $\boldsymbol\delta^{(3)} = \mathbf{a}^{(3)} - \mathbf{y}$):

  • $\partial J/\partial \mathbf{b}^{(2)} = \boldsymbol\delta^{(3)} = (-0.745, 0.71, 0.035)$ when target = class 1.
  • $\partial J/\partial w_{14}^{(2)} = \delta_1^{(3)} \cdot a_4^{(2)} = (-0.745)(0.72) = -0.536$.
  • $\delta_4^{(2)}$ via $(W^{(2)\top}\boldsymbol\delta^{(3)}) \odot \text{ReLU}'(\mathbf{z}^{(2)}) = -0.170$ for that element.

Batch GDUAS 2021 · Soal 2

2. Linear regression — one epoch of batch GD

Data: $(3,2), (7,17), (11,2), (15,17), (19,2), (23,17)$. Model $\hat y = w_0 + w_1 x$ with $w_0 = -1.75$, $w_1 = 1.25$, $\alpha = 0.01$, loss $L = \tfrac{1}{2}\sum (\hat y - y)^2$. After one full epoch of batch GD, find:

  • Initial loss $L$
  • New $w_0$ and $w_1$
Solution
  1. Predictions: $\hat y = -1.75 + 1.25x$ for $x = 3, 7, 11, 15, 19, 23$ gives $\hat y = 2, 7, 12, 17, 22, 27$.
  2. Errors $(\hat y - y)$: $0, -10, 10, 0, 20, 10$.
  3. $L = \tfrac{1}{2}(0 + 100 + 100 + 0 + 400 + 100) = \tfrac{1}{2}(700) = \boxed{350}$.
  4. $\partial L/\partial w_0 = \sum \text{err} = 0 - 10 + 10 + 0 + 20 + 10 = 30$.
  5. $\partial L/\partial w_1 = \sum \text{err} \cdot x = 0(3) + (-10)(7) + 10(11) + 0(15) + 20(19) + 10(23) = 0 - 70 + 110 + 0 + 380 + 230 = 650$.
  6. Updates: $w_0 = -1.75 - 0.01(30) = \boxed{-2.05}$, $w_1 = 1.25 - 0.01(650) = \boxed{-5.25}$.
SVMUAS 2021 · Soal 3

3. Hard-margin SVM — which point matters?

You have a hard-margin linear SVM. Removing which type of point changes the decision boundary?

  • A. Any data point in the dataset.
  • B. Only points far from the boundary.
  • C. Only support vectors (points on or closest to the margin).
  • D. Only points of the minority class.
Solution

Removing a point changes the boundary only if it's a support vector — a point on the margin. Non-SV points contribute zero to the SVM solution; deleting them changes nothing.

EvaluationUAS 2021 · Soal 4

4. Precision / Recall / F1

Stratified 20% test split on 1000 samples (with 1000 positives total? Re-read: positive class has 1000 → test has 200 positives). FN = 100, FP = 150. Compute precision, recall, and F1.

Solution
  1. Test positives = 200, FN = 100 ⇒ TP = 200 − 100 = 100.
  2. Precision = $\dfrac{TP}{TP + FP} = \dfrac{100}{100 + 150} = \dfrac{100}{250} = \boxed{0.40}$.
  3. Recall = $\dfrac{TP}{TP + FN} = \dfrac{100}{200} = \boxed{0.50}$.
  4. F1 = $\dfrac{2 \cdot 0.40 \cdot 0.50}{0.90} = \dfrac{0.40}{0.90} \approx \boxed{0.44}$.

ImbalancedUAS 2021 · Soal 7

7. Misleading metric on imbalanced data

A dataset has 99% negative class. A classifier always predicts negative. Which metric reports the misleadingly high "99%" success?

  • A. Accuracy
  • B. Recall (on positive class)
  • C. F1 score
  • D. Precision
Solution

Accuracy = (TN + TP) / N. With everything predicted as negative, accuracy = 99% even though the model is useless. Recall, F1, and precision would all be 0 for the positive class.

Hyperplane $5x_1 + x_2 - 3 = 0$. So $\mathbf{w} = (5, 1)$. Compute the margin.

Solution

10. SMOTE neighbor selection

SMOTE picks nearest neighbors of the same class to interpolate. For two minority (green) instances $a$ and $h$ with mixed-class surroundings, identify the valid SMOTE neighbors.

Solution

SMOTE only interpolates between same-class neighbors.

k-NN + JaccardEssay 1

E1. k-NN with Jaccard distance

Binary features. Jaccard similarity $J = a/(a+b+c)$, distance $d = 1 - J$. Rama's profile = (game=1, film=0, fruit=1, vegetable=1). Distances to friends:

Andy 9/12, Budi 6/12, Caca 8/12, Didi 9/12, Eri 8/12, Fifi 6/12.

With $k = 3$, what is the predicted class?

Solution

Two closest are Budi & Fifi (both 6/12). The third candidate is tied between Caca and Eri (both 8/12).

Majority voting: Budi(Minus), Fifi(Tidak). The third vote breaks the tie:

  • If third = Caca(Minus) → prediction = Minus.
  • If third = Eri(Tidak) → prediction = Tidak.

The answer depends on tie-breaking. Mention both possibilities in the exam.

SVM FullEssay 2

E2. SVM — boundary, margin, classification, support vectors

SVM with $\mathbf{w} = (2, 3)$, $b = -8$, and $\delta = 1$. Find:

  1. Equations of $H_0, H_1, H_2$.
  2. Margin.
  3. Classify points $(2, 2)$ and $(2, 0)$.
  4. Give an example support vector on each margin.
Solution
  1. Boundaries: $H_0: 2x + 3y - 8 = 0$; $H_2: 2x + 3y - 8 = +1$; $H_1: 2x + 3y - 8 = -1$.
  2. Margin $= \dfrac{2}{\|\mathbf{w}\|} = \dfrac{2}{\sqrt{4+9}} = \dfrac{2}{\sqrt{13}} \approx 0.55$.
  3. Classification:
    • $f(2,2) = 4 + 6 - 8 = +2 > 0$ → one class.
    • $f(2,0) = 4 + 0 - 8 = -4 < 0$ → other class.
    Different signs → different classes.
  4. Support vectors sit on $H_1$ or $H_2$. Example: $(3.5, 0)$ satisfies $2(3.5) + 3(0) - 8 = -1$ → on $H_1$. $(1.5, 2)$ satisfies $2(1.5) + 3(2) - 8 = +1$ → on $H_2$.
Logistic RegressionEssay 3

E3. Logistic regression — predict + 1 GD ascent step

(a) Weights $\mathbf{w} = (0.1, 0.2, 0.3)$ (with bias), input $\mathbf{x} = (1, 70, 90)$. Compute $P(y=1 \mid \mathbf{x})$.

(b) One gradient-ascent iteration for $w_1$ only, with $w_0 = 0, w_1 = 1, w_2 = -1$, $\alpha = 0.01$. Data: $(x_1, x_2, y) = (2,1,1), (0,4,0), (3,3,0), (3,1,1)$.

Solution

(a) $z = 0.1 + 0.2(70) + 0.3(90) = 0.1 + 14 + 27 = 41.1$. $P(y=1) = 1/(1 + e^{-41.1}) \approx 1$. Pretty much certain.

(b) Update rule: $w_1 \leftarrow w_1 + \alpha \sum_i (y_i - h(\mathbf{x}_i)) x_{1,i}$.

  • $(2,1, y=1)$: $z = 0 + 2 - 1 = 1$, $h = \sigma(1) = 0.731$, contribution = $(1 - 0.731)(2) = 0.537$.
  • $(0,4, y=0)$: $z = -4$, $h \approx 0.018$, contribution = $(0 - 0.018)(0) = 0$.
  • $(3,3, y=0)$: $z = 0$, $h = 0.5$, contribution = $(0 - 0.5)(3) = -1.5$.
  • $(3,1, y=1)$: $z = 2$, $h = 0.881$, contribution = $(1 - 0.881)(3) = 0.357$.

$\partial \ell / \partial w_1 = 0.537 + 0 - 1.5 + 0.357 = -0.6045$.

New $w_1 = 1 + 0.01 \cdot (-0.6045) = \boxed{0.9939}$.

Imbalanced MLEssay 4

E4. Critique a bad imbalanced-classification pipeline

A student applies SMOTE to the entire dataset, then trains and evaluates on the resulting data via plain cross-validation. ROC AUC = 0.95. Identify the two main mistakes.

Solution
  1. Evaluating on synthetic data. SMOTE/oversampling should be applied only on the training fold, not on the test or whole dataset. Evaluating on oversampled data leaks information (synthetic test points may be interpolations of training points!) and gives misleadingly high scores. ROC AUC 0.95 is too optimistic.
  2. No stratified cross-validation. On imbalanced data, plain CV may produce folds with very few minority samples — or none at all. Stratified CV preserves class proportions per fold and is essential.

Fix: use the imbalanced-learn Pipeline that applies SMOTE inside each training fold only, combined with StratifiedKFold. Test on real (non-resampled) data.

E5. Cross-entropy + hidden-layer backprop

(a) Softmax output $[0.7, 0.2, 0.1]$, target $[0, 0, 1]$. Compute CE loss.

True or False: "Naive Bayes has higher variance than bias."

  • A. TRUE
  • B. FALSE
Solution

FALSE. Naive Bayes has high bias (due to the strong conditional independence assumption) and low variance. It's the opposite of the claim.

RegularizationUAS 2023 · Soal 2

2. Which statement about L1/L2 is INCORRECT?

  • A. If $\lambda$ on L2 is very large, the model tends to overfit.
  • B. L1 produces sparsity by driving weights to zero.

A is incorrect. Large $\lambda$ → small weights → simpler model → underfit, not overfit.

ClusteringUAS 2023 · Soal 5

5. K-Means vs agglomerative — incorrect claims

Identify the INCORRECT statements about K-Means vs hierarchical agglomerative.

  • A. A given dataset can be clustered by either algorithm; the two are not mutually exclusive.
  • B. For $N \gg k$, k-means storage is larger than agglomerative.
  • C. For $N \gg k$, k-means storage $O(N)$ is smaller than agglomerative $O(N^2)$.
  • D. Only one of the two algorithms can handle a given dataset.

K-Means updateClustering · Soal 1

1. K-Means centroid update

Current clusters: $C_1 = \{P_2, P_5, P_6\}$ where $P_2 = (2,0,1)$, $P_5 = (-2,4,0)$, $P_6 = (0,2,2)$. Compute the next centroid of $C_1$.

Solution

Centroid = element-wise mean:

  • $x$: $(2 + (-2) + 0)/3 = 0$
  • $y$: $(0 + 4 + 2)/3 = 2$
  • $z$: $(1 + 0 + 2)/3 = 1$

New $C_1$ centroid = $\boxed{(0, 2, 1)}$.

Complete linkageClustering · Soal 2

2. Agglomerative clustering — complete linkage

After merging $A$ and $B$ into cluster $\{AB\}$, complete-linkage uses $d(\{AB\}, X) = \max(d(A,X), d(B,X))$. Given:

  • $d(A,C) = 0.51, d(B,C) = 0.25$ → $r$?
  • $d(A,D) = 0.84, d(B,D) = 0.16$ → $s$?
  • $d(A,E) = 0.28, d(B,E) = 0.77$ → $t$?
  • $d(A,F) = 0.38, d(B,F) = 0.61$ → $u$?
Solution

Take the maximum each time:

  • $r = \max(0.51, 0.25) = \boxed{0.51}$
  • $s = \max(0.84, 0.16) = \boxed{0.84}$
  • $t = \max(0.28, 0.77) = \boxed{0.77}$
  • $u = \max(0.38, 0.61) = \boxed{0.61}$

Continuing complete-linkage agglomerative produces the corresponding dendrogram (Figure 3 in the original exam).
