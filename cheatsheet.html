The Cheatsheet

Every key formula in one printable page. Designed for the night before the exam.

1 — Probability & MLE

Bayes' theorem

$$P(C_i \mid A) = \frac{P(A \mid C_i)\, P(C_i)}{\sum_j P(A \mid C_j)\, P(C_j)}$$

Prior × likelihood ÷ normalizer = posterior.

Law of total probability

$$P(A) = \sum_i P(A \mid C_i)\, P(C_i)$$

Partition $\{C_i\}$ covers $\Omega$.

Likelihood & log-likelihood (i.i.d.)

$$L(\theta) = \prod_i p(x_i \mid \theta),\quad LL(\theta) = \sum_i \log p(x_i \mid \theta)$$

Use log to convert product to sum, avoid underflow.

MLE recipe

  1. Write $LL(\theta) = \sum_i \log p(x_i \mid \theta)$.
  2. $\dfrac{dLL}{d\theta} = 0$, solve.
  3. Verify maximum (2nd derivative).

Common MLE closed forms

  • Bernoulli($p$): $\hat p = k/N$ (successes/trials).
  • Binomial($n, p$): $\hat p = (\sum_i x_i)/(Nn)$.
  • Categorical / Multinomial: $\hat\theta_k = N_k / N$ (relative frequency).
  • Poisson($\lambda$): $\hat\lambda = \bar x$.
  • Geometric($p$): $\hat p = 1/\bar x$.
  • Normal $\mathcal{N}(\mu, \sigma^2)$: $\hat\mu = \bar x$, $\hat\sigma^2 = \frac{1}{N}\sum_i (x_i - \hat\mu)^2$.
  • Exponential($\lambda$): $\hat\lambda = 1/\bar x$.
  • Uniform $[a, b]$: $\hat a = \min_i x_i$, $\hat b = \max_i x_i$.

Naive Bayes

$$\hat y = \arg\max_y P(y) \prod_j P(x_j \mid y)$$

Conditional independence given $y$. High bias, low variance.

Gaussian NB (numeric features)

$$P(x \mid y) = \tfrac{1}{\sqrt{2\pi\sigma_y^2}} e^{-(x-\mu_y)^2/(2\sigma_y^2)}$$

Estimate $\mu_y, \sigma_y$ per class from training data.

2 — Linear / Logistic / Softmax

Master decision table

TaskActivationLossTarget
RegressionIdentityMSE/SSE$y \in \mathbb{R}$
Binary classificationSigmoidBCE$y \in \{0,1\}$
Multi-class (one-hot)SoftmaxCCE$\mathbf{y} \in \{0,1\}^K$
Multi-labelSigmoid (each)Sum of BCEs$\mathbf{y} \in \{0,1\}^K$

Linear regression

$$h_\mathbf{w}(\mathbf{x}) = \mathbf{w}^\top \mathbf{x}, \quad L = \tfrac{1}{2}\|\mathbf{Xw} - \mathbf{y}\|_2^2$$ $$\boxed{\mathbf{w}^* = (\mathbf{X}^\top\mathbf{X})^{-1}\mathbf{X}^\top \mathbf{y}}$$

Closed form (normal equation).

Logistic regression

$$h = \sigma(\mathbf{w}^\top\mathbf{x}),\; L = -\sum_i [y_i \log h_i + (1-y_i)\log(1-h_i)]$$

No closed form. Sigmoid: $\sigma'(z) = \sigma(z)(1 - \sigma(z))$.

Softmax regression

$$s_k = \frac{e^{\mathbf{w}_k^\top \mathbf{x}}}{\sum_\ell e^{\mathbf{w}_\ell^\top \mathbf{x}}},\quad L = -\sum_i \mathbf{y}_i^\top \log \mathbf{s}_i$$

$\mathbf{W} \in \mathbb{R}^{K \times D}$ (K rows, D cols). Total params $= K(D+1)$.

Regularization

  • L1 (Lasso): $+\lambda \sum_j |w_j|$ → sparsity (feature selection).
  • L2 (Ridge): $+\lambda \sum_j w_j^2$ → smooth shrinkage.
  • Large $\lambda$ → simpler model → underfit (not overfit!).
  • $\lambda = 0$ → no regularization.

3 — Gradient descent

The universal pattern

$$\nabla_\mathbf{w} L = \sum_i (\text{prediction}^{(i)} - \text{target}^{(i)}) \cdot \mathbf{x}^{(i)\top}$$

True for linear, logistic, and softmax. Only "prediction" changes ($\mathbf{w}^\top\mathbf{x}$, $\sigma(\cdot)$, softmax respectively).

Update rule

$$\mathbf{w} := \mathbf{w} - \alpha (\nabla_\mathbf{w} L)^\top$$

For maximizing $\ell$: ascent ($+\alpha$). For minimizing NLL: descent ($-\alpha$). Same update, sign cancels.

With regularization

  • L2: $\nabla J = \nabla L + 2\lambda \mathbf{w}$.
  • L1: $\nabla J = \nabla L + \lambda\, \text{sign}(\mathbf{w})$.

Variants

  • Batch GD — 1 update per epoch. Accurate, slow.
  • SGD — 1 update per example. Fast, noisy.
  • Mini-batch — best compromise (modern default).

Stopping criteria

  • Max iterations reached.
  • $\|\mathbf{w}' - \mathbf{w}\|_2^2 \leq \varepsilon^2$.
  • $|L' - L| \leq \varepsilon$.
  • $\|\nabla L\| \leq \varepsilon$.

4 — Clustering

Distance metrics

  • Euclidean (L2): $\sqrt{\sum_i (u_i - v_i)^2}$
  • Manhattan (L1): $\sum_i |u_i - v_i|$
  • Chebyshev (L∞): $\max_i |u_i - v_i|$
  • Cosine distance: $1 - \frac{\mathbf{u}^\top\mathbf{v}}{\|\mathbf{u}\|\|\mathbf{v}\|}$
  • Hamming: $\sum_i \mathbb{1}[u_i \neq v_i]$ (categorical)
  • Jaccard: $1 - \frac{|A \cap B|}{|A \cup B|}$ (binary sets)

K-Means

$$\text{SSE} = \sum_j \sum_{\mathbf{x} \in C_j} \|\mathbf{x} - \mathbf{m}_j\|_2^2$$

Assign nearest centroid → recompute mean → repeat. $O(t \cdot k \cdot n)$. Sensitive to outliers; use K-Medoids for robustness.

Linkage methods (hierarchical)

  • Single: $\min(d(A,X), d(B,X))$ → chains.
  • Complete: $\max(d(A,X), d(B,X))$ → compact.
  • Average: $\text{avg}(d(A,X), d(B,X))$ → balanced.
  • Ward: minimize within-cluster SSE.

Silhouette

$$S(o) = \frac{b(o) - a(o)}{\max(a(o), b(o))}$$
  • $a(o)$ = mean intra-cluster distance.
  • $b(o)$ = min mean distance to another cluster.
  • $S \in [-1, 1]$. Pick $k$ at peak.
  • $\geq 0.7$ strong, $0.5–0.7$ moderate, $\leq 0.25$ no structure.

External evaluation (pairs)

  • Precision = $\frac{TP}{TP + FP}$
  • Recall = $\frac{TP}{TP + FN}$
  • F1 = $\frac{2PR}{P+R}$
  • Rand Index = $\frac{TP + TN}{\text{all pairs}}$
  • Jaccard = $\frac{TP}{TP + FP + FN}$

K-Means vs K-Medoids

  • K-Means uses mean → sensitive to outliers, $O(n)$.
  • K-Medoids uses an actual data point → outlier-robust, $O(n^2)$.

5 — Neural networks

Forward pass (one layer)

$$\mathbf{p}^{(\ell)} = \mathbf{W}^{(\ell)} \mathbf{a}^{(\ell-1)} + \mathbf{b}^{(\ell)}$$ $$\mathbf{a}^{(\ell)} = g^{(\ell)}(\mathbf{p}^{(\ell)})$$

Parameter dimensions

  • $\mathbf{W}^{(\ell)}$ has shape $D_\ell \times D_{\ell-1}$ (output rows × input cols).
  • $\mathbf{b}^{(\ell)}$ has shape $D_\ell$.
  • Total params $= \sum_\ell (D_\ell D_{\ell-1} + D_\ell)$.

Activation functions & derivatives

Function$g(u)$$g'(u)$
Identity$u$$1$
Sigmoid$1/(1+e^{-u})$$g(u)(1-g(u))$
Tanh$\tanh(u)$$1 - \tanh^2(u)$
ReLU$\max(0, u)$$1$ if $u>0$, else $0$
Leaky ReLU$u$ or $\alpha u$$1$ or $\alpha$
Softplus$\ln(1 + e^u)$$\sigma(u)$ ← sigmoid!
ELU$u$ or $\alpha(e^u-1)$$1$ or $\alpha e^u$
Softmax$e^{u_k}/\sum_\ell e^{u_\ell}$$s_k(\mathbb{1}_{k=j} - s_j)$

Memorize: derivative of softplus = sigmoid. Comes up often in UAS.

Loss derivatives

  • MSE: $\partial L/\partial \mathbf{a}^{(L)} = \mathbf{a}^{(L)} - \mathbf{y}$.
  • BCE: $\partial L/\partial a^{(L)} = (a^{(L)} - y)/(a^{(L)}(1-a^{(L)}))$.
  • CCE: $\partial L/\partial a_k^{(L)} = -y_k/a_k^{(L)}$.

⚡ The universal shortcut

For these three pairings ONLY:

$$\overline{\mathbf{p}^{(L)}} = \mathbf{a}^{(L)} - \mathbf{y}$$
  • Identity + MSE
  • Sigmoid + BCE
  • Softmax + CCE

This is why they're the standard pairs.

6 — Backpropagation recipe

Step-by-step backward pass

Step 1. Forward: compute $\mathbf{p}^{(\ell)}, \mathbf{a}^{(\ell)}$ for $\ell = 1, \ldots, L$.

Step 2. $\overline{L} = 1$.

Step 3. Output: $\overline{\mathbf{p}^{(L)}} = \mathbf{a}^{(L)} - \mathbf{y}$ (if matching shortcut), or compute $\overline{\mathbf{a}^{(L)}} \odot g^{(L)\prime}(\mathbf{p}^{(L)})$.

Step 4. For $\ell = L, L-1, \ldots, 1$:

  • $\overline{\mathbf{W}^{(\ell)}} = \overline{\mathbf{p}^{(\ell)}} (\mathbf{a}^{(\ell-1)})^\top$ (outer product).
  • $\overline{\mathbf{b}^{(\ell)}} = \overline{\mathbf{p}^{(\ell)}}$.
  • $\overline{\mathbf{a}^{(\ell-1)}} = (\mathbf{W}^{(\ell)})^\top \overline{\mathbf{p}^{(\ell)}}$.
  • $\overline{\mathbf{p}^{(\ell-1)}} = \overline{\mathbf{a}^{(\ell-1)}} \odot g^{(\ell-1)\prime}(\mathbf{p}^{(\ell-1)})$.

Step 5. Update: $\mathbf{W}^{(\ell)} := \mathbf{W}^{(\ell)} - \alpha \overline{\mathbf{W}^{(\ell)}}$, similarly for $\mathbf{b}$.

Memory trick

Forward: multiply by $\mathbf{W}$ → add $\mathbf{b}$ → apply $g$.

Backward unwinds in reverse:

  1. Multiply elementwise by $g'$.
  2. Copy to $\overline{\mathbf{b}}$.
  3. Outer-product with $\mathbf{a}^{(\ell-1)}$ for $\overline{\mathbf{W}}$.
  4. Multiply by $\mathbf{W}^\top$ for prev layer.

Output activation by task

  • Regression → Identity + MSE.
  • Binary → Sigmoid + BCE.
  • Multi-class (one-hot) → Softmax + CCE.
  • Multi-label → Sigmoid each + sum of BCEs.

7 — Final exam tips

Common gotchas
  • Always check the metric (Euclidean vs Manhattan) before computing distances — answers can differ drastically.
  • Softmax $\mathbf{W}$ has shape $K \times D$, not $D \times K$. Always: rows = outputs, cols = inputs.
  • Large regularization $\lambda$ → simpler model → underfit, not overfit.
  • L1 for feature selection, L2 for general smoothing.
  • GD minimizes; for maximizing log-likelihood, use gradient ascent — or equivalently GD on NLL.
  • Softmax output is always in $(0, 1)$ and sums to 1.
  • Derivative of softplus = sigmoid. Often tested in MLP problems.
  • NB is conditionally independent (given $y$), not unconditionally.
  • SMOTE goes only on the training fold, never on test data — and use stratified CV.
  • Accuracy is useless on imbalanced data. Use F1, ROC AUC, balanced metrics.
  • SVM: only support vectors change the boundary; non-SV points contribute nothing.
  • Decision trees are NOT trained with gradient descent (greedy splitting).
The three things you'll use most

1. The universal gradient pattern for linear/logistic/softmax:

$$\nabla_\mathbf{w} L = \sum_i (\text{prediction}^{(i)} - \text{target}^{(i)}) \cdot \mathbf{x}^{(i)\top}$$

2. The universal output-layer backprop shortcut (for the standard activation+loss pairs):

$$\overline{\mathbf{p}^{(L)}} = \mathbf{a}^{(L)} - \mathbf{y}$$

3. Bayes' theorem — the foundation of generative modeling:

$$P(y \mid \mathbf{x}) = \frac{P(\mathbf{x} \mid y) P(y)}{P(\mathbf{x})}$$
