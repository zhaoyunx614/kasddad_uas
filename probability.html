Likelihood & Bayes

Probability axioms, conditional probability, Bayes' theorem, and maximum likelihood estimation — the foundations everything else builds on.

1. Basic probability concepts

The sample space $\Omega$ is the set of all possible outcomes. An event is a subset of the sample space. The probability $P(A)$ states how likely event $A$ occurs, with $0 \leq P(A) \leq 1$.

The three axioms

  • Non-negativity: $P(A) \geq 0$ for all $A \subseteq \Omega$.
  • Normalization: $P(\Omega) = 1$ — something in the sample space must happen.
  • Additivity (for mutually exclusive events): $P(A_1 \cup A_2 \cup \cdots) = P(A_1) + P(A_2) + \cdots$

Two derived rules you use all the time:

Derived rules $$P(A \cup B) = P(A) + P(B) - P(A \cap B), \qquad P(A^c) = 1 - P(A).$$
Where: $\Omega$ = sample space; $A, B$ = events (subsets of $\Omega$); $A \cup B$ = "A or B"; $A \cap B$ = "A and B"; $A^c$ = complement of $A$ (i.e. $A$ does not occur).
🏀 NBA Example — Curry's free throw

Formula (relative frequency): $P(E) = \frac{\#\text{occurrences of }E}{\#\text{trials}}$

Setup. Stephen Curry: 9234 free throws attempted, 8543 made. Sample space $\Omega = \{\text{Make}, \text{Miss}\}$.

Substitution. $P(\text{Make}) = \dfrac{8543}{9234} \approx 0.925.$ The probability is derived from observation data — the relative frequency approach.

2. Conditional probability

Conditional probability $$P(A \mid B) = \frac{P(A, B)}{P(B)}, \quad P(B) > 0.$$
Where: $P(A \mid B)$ = probability of $A$ given that $B$ has occurred; $P(A, B) = P(A \cap B)$ = joint probability (both happen); $P(B)$ = marginal probability of $B$ (must be positive so we don't divide by zero).

From this we derive the product rule, which says the same joint can be factored two ways:

Product rule $$P(A, B) = P(A \mid B)P(B) = P(B \mid A)P(A).$$

3. Law of total probability

If $C_1, \ldots, C_n$ form a partition of $\Omega$ (mutually exclusive and exhaustive):

Law of total probability $$P(A) = \sum_{i=1}^{n} P(A \mid C_i) P(C_i).$$
Where: $C_1, \ldots, C_n$ partition the sample space; $P(A \mid C_i)$ = probability of $A$ inside scenario $C_i$; $P(C_i)$ = probability of being in scenario $C_i$. The formula adds contributions from every scenario.
🏀 NBA Example — LeBron three-point shooting

Formula: $P(\text{Make}) = \sum_i P(\text{Make} \mid \text{Zone}_i)\, P(\text{Zone}_i)$

Setup. Make rates: open look 40%, contested 30%, transition 55%. Zone distribution: 50% open, 30% contested, 20% transition.

Substitution.

$$P(\text{Make}) = (0.40)(0.50) + (0.30)(0.30) + (0.55)(0.20) = 0.20 + 0.09 + 0.11 = 0.40.$$

LeBron has an overall 40% make rate considering all zones.

4. Bayes' theorem

Bayes' theorem (the centerpiece) $$P(Y\!=\!y \mid X\!=\!x) = \frac{P(X\!=\!x \mid Y\!=\!y)\, P(Y\!=\!y)}{P(X\!=\!x)} = \frac{P(X\!=\!x \mid Y\!=\!y)\, P(Y\!=\!y)}{\sum_{y'} P(X\!=\!x \mid Y\!=\!y')\, P(Y\!=\!y')}.$$
Where: $Y$ = hidden quantity we want to infer (disease state, true class); $X$ = observed data; $P(Y\!=\!y)$ = prior (belief about $Y$ before seeing data); $P(X\!=\!x \mid Y\!=\!y)$ = likelihood (how well hypothesis $y$ explains observed $x$); $P(Y\!=\!y \mid X\!=\!x)$ = posterior (updated belief); $P(X\!=\!x)$ = normalization constant (sum over every possible $y'$ so probabilities total 1).
🏀 NBA Example — ACL injury diagnosis

Setup. Sensitivity $P(\text{Positive} \mid \text{ACL}) = 0.95$; false-positive rate $P(\text{Positive} \mid \text{No ACL}) = 0.10$; prior $P(\text{ACL}) = 0.02$.

Step 1 — denominator (law of total probability):

$$P(\text{Positive}) = (0.95)(0.02) + (0.10)(0.98) = 0.019 + 0.098 = 0.117.$$

Step 2 — apply Bayes:

$$P(\text{ACL} \mid \text{Positive}) = \frac{(0.95)(0.02)}{0.117} \approx 0.162.$$

Although the test is accurate, because ACL injuries are rare, only ≈16% of positive results actually correspond to ACL injuries. This is the famous base-rate effect.

5. Likelihood and MLE

The likelihood of a parameter $\theta$ given observed data $x$ is just the probability of that data treating $\theta$ as the variable:

Likelihood (single observation) $$L(\theta \mid X\!=\!x) = P(X\!=\!x \mid \theta).$$
Where: Numerically equal to the probability, but we treat $x$ as fixed and vary $\theta$. The likelihood is a function of $\theta$, not of $x$.

For an i.i.d. (independent & identically distributed) dataset $\mathcal{D} = \{x_1, \ldots, x_N\}$:

Likelihood & log-likelihood for an i.i.d. sample $$L(\theta) = \prod_{i=1}^{N} p(x_i \mid \theta), \qquad LL(\theta) = \log L(\theta) = \sum_{i=1}^{N} \log p(x_i \mid \theta).$$

Maximum Likelihood Estimation (MLE) finds the $\theta$ that makes the observed data most probable:

$$\hat\theta_{\text{MLE}} = \arg\max_\theta L(\theta) = \arg\max_\theta LL(\theta) = \arg\min_\theta \text{NLL}(\theta).$$

where $\text{NLL}(\theta) = -LL(\theta)$ is the negative log-likelihood — what we usually minimize in practice.

6. Why log-likelihood?

The numerical reason for using log

Problem. Since each $p(x_i \mid \theta) < 1$, multiplying many small values shrinks toward zero (numerical underflow). Example: if $N = 1000$ and $p \approx 0.5$, then

$$L(p) = 0.5^{1000} \approx 7.8 \times 10^{-31} \;-\; \text{too small for computer precision.}$$

Solution. The logarithm identity $\log(a \cdot b) = \log a + \log b$ converts multiplication into addition. For the example above, $LL = \log(7.8 \times 10^{-31}) \approx -69.31$ — safely handled.

Since log is monotonically increasing, the optimum doesn't change:

$$\arg\max_\theta L(\theta) = \arg\max_\theta LL(\theta).$$

7. Distribution catalog (PMF/PDF + MLE)

For each distribution we list: the PMF/PDF, the log-likelihood for an i.i.d. sample of size $N$, and the closed-form MLE.

Discrete distributions

1 · Bernoulli(p) — single yes/no trial, $x \in \{0,1\}$ $$p(x \mid p) = p^{x}(1-p)^{1-x}, \qquad LL(p) = k \log p + (N-k)\log(1-p),$$ $$\boxed{\;\hat p_{\text{MLE}} = \frac{k}{N}\;}$$

where $k = \sum_i x_i$ = number of successes. The MLE = relative frequency.

2 · Binomial(n, p) — n Bernoulli trials counted, $x \in \{0,\ldots,n\}$ $$p(x \mid n, p) = \binom{n}{x} p^x (1-p)^{n-x}, \qquad \boxed{\;\hat p_{\text{MLE}} = \frac{\sum_i x_i}{Nn}\;}$$

Total successes divided by total trials. The $\binom{n}{x}$ factor doesn't depend on $p$ and drops out.

3 · Categorical($\boldsymbol\theta$) — one trial with K outcomes, one-hot $\mathbf{x}$ $$p(\mathbf{x} \mid \boldsymbol\theta) = \prod_{k=1}^{K} \theta_k^{x_k}, \quad \sum_k \theta_k = 1, \qquad \boxed{\;\hat\theta_k = \frac{N_k}{N}\;}$$

where $N_k$ = number of observations in category $k$. MLE = relative frequency.

4 · Multinomial(n, $\boldsymbol\theta$) — n categorical trials counted by category $$p(\mathbf{x} \mid n, \boldsymbol\theta) = \frac{n!}{x_1! \cdots x_K!} \prod_{k=1}^{K} \theta_k^{x_k}, \qquad \boxed{\;\hat\theta_k = \frac{\sum_i x_{i,k}}{Nn}\;}$$
5 · Poisson($\lambda$) — counts in a fixed interval, $x \in \{0,1,2,\ldots\}$ $$p(x \mid \lambda) = \frac{\lambda^x e^{-\lambda}}{x!}, \qquad LL(\lambda) = \sum_i \left[x_i \log\lambda - \lambda - \log x_i!\right],$$ $$\boxed{\;\hat\lambda_{\text{MLE}} = \bar x = \tfrac{1}{N}\sum_i x_i\;}$$

The MLE is the sample mean. (The $\log x_i!$ term is constant in $\lambda$ and drops out.)

6 · Geometric(p) — trials until first success, $x \in \{1,2,\ldots\}$ $$p(x \mid p) = (1-p)^{x-1} p, \qquad \boxed{\;\hat p_{\text{MLE}} = \frac{1}{\bar x}\;}$$

Continuous distributions

7 · Normal $\mathcal{N}(\mu, \sigma^2)$ $$p(x \mid \mu, \sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right),$$ $$\boxed{\;\hat\mu_{\text{MLE}} = \bar x, \quad \hat\sigma^2_{\text{MLE}} = \tfrac{1}{N}\sum_i (x_i - \hat\mu)^2\;}$$

Note: MLE variance divides by $N$, not $N-1$ — a famously "biased" estimator.

8 · Exponential($\lambda$) — time between Poisson events, $x \geq 0$ $$p(x \mid \lambda) = \lambda e^{-\lambda x}, \qquad LL(\lambda) = N\log\lambda - \lambda\sum_i x_i,$$ $$\boxed{\;\hat\lambda_{\text{MLE}} = \frac{1}{\bar x} = \frac{N}{\sum_i x_i}\;}$$
9 · Uniform on $[a, b]$ $$p(x \mid a, b) = \frac{1}{b-a} \text{ for } x \in [a,b], \;0 \text{ otherwise}.$$ $$\boxed{\;\hat a_{\text{MLE}} = \min_i x_i, \quad \hat b_{\text{MLE}} = \max_i x_i\;}$$

The endpoints are the most extreme observations.

General two-step MLE recipe
  1. Write the log-likelihood $LL(\theta) = \sum_i \log p(x_i \mid \theta)$.
  2. Take $\dfrac{dLL}{d\theta}$, set to zero, solve for $\theta$.
  3. Verify it's a maximum (second derivative or boundary check).

8. Worked examples

Example A — MLE for a Bernoulli coin

A coin is tossed 5 times, results $\{H, H, H, H, T\}$. Bernoulli with $p = P(H)$:

$$L(p) = p^{4}(1-p)^{1}, \qquad LL(p) = 4\log p + \log(1-p), \qquad \frac{dLL}{dp} = \frac{4}{p} - \frac{1}{1-p}.$$

Setting $\dfrac{dLL}{dp} = 0$:

$$\frac{4}{p} = \frac{1}{1-p} \;\Longrightarrow\; 4(1-p) = p \;\Longrightarrow\; \hat p = \frac{4}{5} = 0.8.$$

This equals the relative frequency $\#H/(\#H + \#T)$, as expected.

🏀 NBA Example — Klay Thompson's FT%

Klay shoots $n = 8$ free throws, makes $k = 7$.

$$L(p) = p^{7}(1-p)^{1}, \qquad LL(p) = 7\log p + \log(1-p),$$ $$\frac{dLL}{dp} = \frac{7}{p} - \frac{1}{1-p} = 0 \;\Longrightarrow\; \hat p_{\text{MLE}} = \frac{7}{8} = 0.875.$$

His FT% estimate from this small sample is 87.5%.

Example B — When MLE overfits (small sample)

Tutorial problem — coin all heads

A coin is tossed 3 times, results $\{H,H,H\}$. Then $L(\theta) = \theta^3$, $LL(\theta) = 3\log\theta$, and $\dfrac{dLL}{d\theta} = \dfrac{3}{\theta} > 0$ everywhere — $LL$ is monotonically increasing on $[0,1]$, so the maximum is at $\theta = 1$.

Mathematically: $\hat\theta_{\text{MLE}} = 1$. Physically: nonsense — a real coin doesn't have $P(H) = 1$. Lesson: MLE trusts small samples too much. Use priors (MAP) or get more data.

🏀 NBA analogue

Giannis takes 3 free throws, makes all 3. MLE says $\hat p_{\text{MLE}} = 3/3 = 1.0$ (100%). But his career FT% is ≈ 70%. Three samples isn't enough to override a strong prior.

9. Naive Bayes classifier

The Naive Bayes classifier is Bayes' theorem with an extra assumption: features are conditionally independent given the class.

The Naive Bayes assumption $$P(x_1, x_2, \ldots, x_d \mid y) = \prod_{j=1}^{d} P(x_j \mid y).$$

This is the assumption that features are independent given $y$. They are not assumed to be unconditionally independent — that would be much stronger.

Combined with Bayes' theorem, the classifier picks:

$$\hat y = \arg\max_y P(y) \prod_{j=1}^{d} P(x_j \mid y).$$

Mixed feature types (categorical + Gaussian)

In practice, datasets mix categorical features (e.g. "rainfall: low/medium/high") with numeric ones (e.g. "temperature in °C"). Naive Bayes handles both:

  • Categorical features: use frequency estimates (often with Laplace smoothing to avoid zero counts).
  • Numeric features: assume a Gaussian per class — $P(x \mid y) = \frac{1}{\sqrt{2\pi\sigma_y^2}}\exp\!\left(-\frac{(x - \mu_y)^2}{2\sigma_y^2}\right)$, with $\mu_y$ and $\sigma_y^2$ estimated from the training data of class $y$.
Tutorial — categorical MLE with 3 classes

Sentiment data $\mathcal{D} = \{\text{Pos}, \text{Pos}, \text{Pos}, \text{Neu}, \text{Neu}, \text{Neg}\}$ (6 points). With parameters $(\theta_1, \theta_2, \theta_3)$ for (Positive, Neutral, Negative):

$$\hat\theta_1 = \tfrac{3}{6} = 0.5, \quad \hat\theta_2 = \tfrac{2}{6} \approx 0.333, \quad \hat\theta_3 = \tfrac{1}{6} \approx 0.167.$$

Maximum likelihood score:

$$L_{\max} = (0.5)^3 (0.333)^2 (0.167)^1 \approx 0.00231.$$

You'll see the full mixed Naive Bayes example (Soal 11 from UAS 2023/2024) worked out step by step in the Exercises page.

Naive Bayes characterization (UAS 2023 Soal 1)

Naive Bayes has high bias (strong conditional independence assumption) and low variance. The claim "variance higher than bias" is FALSE.
